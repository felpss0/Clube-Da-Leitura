# Clube Da Leitura

## Projeto

Desenvolvido durante o curso FullStack da [Academia do Programador](https://www.academiadoprogramador.net/inicio) 2025

## Introdução

Gustavo sempre adorou compartilhar sua vasta coleção de revistas em quadrinhos
com os amigos, mas com o crescimento do Clube da Leitura, a organização manual
se tornou um verdadeiro desafio. Ele precisa anotar cuidadosamente todas as
informações dos participantes.

Foi essa série de desafios que levou seu pai a perceber que a solução estava na
tecnologia, contratando os alunos da Academia do Programador 2025 para
transformar esse sistema manual e propenso a erros em uma aplicação digital que
pudesse finalmente trazer ordem ao Clube da Leitura.

## Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.

## Como Utilizar

####  Clone o Repositorio 

```
git clone https://github.com/felpss0/Clube-Da-Leitura.git
```

#### Navegue até a pasta raiz da soluçao

```
cd Clube-da-leitura
```

#### Restaure as dependencias 

```
dotnet restore
```

#### Execute o Projeto

```
dotnet run
```
